<p align="center">
    <img src="../imgs/6. 룰렛 집고있기.gif" alt="6. 룰렛 집고있기.gif" width="720px">

위는 초록색만 꺼내도록 한 것인데 저런 식의 퍼즐이 필요할 수도 있다.

#### 낙차 없음

<p align="center">
    <img src="../imgs/9, 낙차 없음.gif" alt="9, 낙차 없음.gif" width="720px">
</p>

테스트룸에서 구현은 했으나, 넣지는 않았음.   
있는 드롭만 떨어트려 맞추고, 재생성하지 않은 채 끝나면 드롭을 재생성함.   
그렇기에 맞춰진 채로 드롭이 등장할 수도 있다.
